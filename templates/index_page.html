A gentle introduction

This is Phuong's solution for the DevOps/Data Engineer Challenge which is organized by Haensel AMS GmbH & Inc. To this end, I wrote computer programming with Python, Flask, HTML. The fundamental features of my solution are described as follows.

  1. First, it has four pages, such as Index, two Uploader pages (local- and AWS-hosted APIs), and conclusion remarks. I would like to want you to evaluate my local-hosted API first, then, the AWS-hosted one.
  2. Second, it checks the file size and only sends the 1-megabyte-below file. If the file size is greater than 1 Megabyte, an error will appear. However, one can change this maximum limit by modifying my Python code.
  3. Third, it checks if a file is selected. Otherwise, a warning message will appear as a kind reminder.
  4. Fourth, it sends three types of file formats, including 'doc','xls','pdf'. However, One can extend the type of file format by modifying my underlying HTML codes.If the uploaded file format is not eligible, a warning message will be generated to remind the user.
  5. Last but not least, it examines whether the uploaded file is forged and dangerous.
